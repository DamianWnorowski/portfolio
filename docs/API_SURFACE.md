### /autorun-all

- `--all` (Purpose)

- `--MaxCycles` (n)
- `--ApplySafeActions` (string)
- `--approved` (safe)

**Usage:**
```bash
- `D:\scripts\autorun_all.ps1 -ApplySafeActions`

Behavior:
- Reads `D:\codex\self_evolution_contract.json`:
  - Honors `auto_loop_enabled` (exits if false).
  - Uses `generation_interval_seconds` as the sleep interval when not overridden.
- Each cycle:
  - Runs `auto_evolve_generation.py` (with `--apply-safe-actions` if requested).
  - Runs `summarize_generation.py` and prints the latest summary.
  - Runs `auto_design_analyzer.py` and writes `D:\MASTER_INDEX\evolution\design_latest.json`.

Flags:
- `-IntervalSeconds` to override the contract interval.
- `-MaxCycles <n>` to run a fixed number of cycles and then exit.
- `-ApplySafeActions` to enable contract-approved safe actions during each generation.

Notes:
- This script does not change the contract or code directly; it drives the existing evolution engine and analyzers in a loop.
- Use `Ctrl+C` or set `auto_loop_enabled` to false in the contract to stop it.
```
